# 📊 Amazon Product Insight Dashboard (Excel)

This project is an interactive Excel dashboard designed to analyze Amazon product data, focusing on pricing, discounts, ratings, and sales performance.

---

## 🚀 Features

- KPI Metrics (Total Sales, Ratings, Discounts)
- Price Range Filtering (Slicers)
- Discount vs Sales Analysis
- Rating vs Discount Insights
- Category-wise Performance Analysis

---

## 🔍 Key Insights

- Product ratings remain stable across discount ranges
- Higher discounts lead to increased sales volume
- No strong correlation between discount % and ratings
- Certain categories show higher engagement than others

---

## 🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Data Visualization
- Data Cleaning
